##  Overview

AgriConnect is a MERN stack web application designed to connect **farmers and buyers** on a single platform.

It helps farmers to showcase crops and buyers to explore agricultural products easily and efficiently.

---

##  Key Features

### 👨‍🌾 Farmer Features

> Add crops
> Edit & delete crops
> Manage dashboard
> Profile management

###  Buyer Features

> Browse crops
> View crop details
> Market exploration

###  Authentication

> User login & registration
> Protected routes

###  Extra Features

> Weather page
> Fully responsive UI

---

##  Tech Stack

### Frontend

* React.js (Vite)
* Tailwind CSS
* Axios

### Backend

* Node.js
* Express.js
* MongoDB + Mongoose

---

##  Application Flow

```
User Registers/Login
        |
  Server validates
        |
 Database (MongoDB)
        |
 Dashboard Access
        |
 Farmer uploads crops / Buyer explores market
```

##  Installation & Setup

### 🔹 Backend Setup

```
cd server
npm install
npm run dev
```

Create `.env` file:

```
MONGO_URI=your_mongodb_connection
PORT=5000
```

---

### 🔹 Frontend Setup

```
cd client
npm install
npm run dev
```

Runs on:

```
http://localhost:5173
```
